// spend-lens

Private Personal Finance Analyzer

Drop your Chase, Bank of America, Amex, or Wells Fargo CSV. See where your money goes. Your data never leaves this browser tab.

Why you can actually trust this

  1. Your CSV is read by your browser's File API — it's never uploaded to any server.
  2. All processing runs in JavaScript on your device, inside this tab.
  3. Open DevTools → Network tab before using. Zero requests will contain your data.
  4. Close this tab, your data is gone. Nothing is saved, nothing is remembered.
  5. The source is open — read it, fork it, run it offline.
No real statement handy? Download a synthetic sample CSV to try it out.
